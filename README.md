# Installation
#### Installing

1. Download the latest .zip file (paynl.ocmod.zip) and upload it into *Extensions* > *Installer* > *Upload*
2. Install the *Pay. Payments for Opencart 4* Extension

##### Setup

1. Go to *Extensions* > *Extensions*
2. Choose extension type *Payments*
3. Install the *Pay.* Extension
5. Click Edit
6. Enter the Token code, API token and Service ID (these can be found in the PAY. Admin Panel --> https://admin.pay.nl/programs/programs)
7. Save the settings

#### Update instructions

##### via manual file upload (keeps all settings)
1. Download the latest .zip file (paynl.ocmod.zip)
2. Unpack the zip
3. upload the zip contents to */extension/paynl*

##### via Extension Manager (removes all settings)
1. Download the latest .zip file (paynl.ocmod.zip)
2. Go to *Extensions* > *Extensions*
3. Uninstall the *Pay.* Extension
4. Go to *Extensions* > *Installer*
5. Uninstall the *Pay. Payments for Opencart 4* Extension
6. Delete the *Pay. Payments for Opencart 4* Extension
7. Upload the new .zip file  (paynl.ocmod.zip)
8. Install the *Pay. Payments for Opencart 4* Extension
